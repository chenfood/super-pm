# super-pm AI Skill

## 项目简介
本项目是一个遵循 **Claude Code / Codex / Trae** 等大模型规范的 **AI Skill** 配置文件。它定义了一套完整的交互设计工作流，旨在让 AI 能够像资深产品经理一样，通过结构化的 PDCA 循环和可视化工具，从零构建高质量的交互方案。

## 核心规范
详细的 AI 指令、约束条件及工作流定义，请参阅 Skill 定义文件：
👉 **[SKILL.md](./SKILL.md)**

## 主要功能
- **AI 角色设定**：定义 AI 作为资深交互设计师的行为准则。
- **结构化工作流**：从 P0 记忆扫描到 P5 原型交付的完整路径。
- **可视化联动**：配套脚本支持 AI 在浏览器中实时渲染交互选项。
- **规范化约束**：强制执行用户画像对齐和设计系统优先原则。

## 如何加载
将此目录添加到您的 AI 助手（如 Trae, Claude Code）的 Skill/Agent 路径中，AI 即可根据 `SKILL.md` 的定义自动触发交互设计能力。
